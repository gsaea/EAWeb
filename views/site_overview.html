Overview

The EA Integration Portal is a tool to provide commonly used strategic, business and technical information to users throughout GSA in order to provide insight into the common direction for GSAIT and to provide working groups with data that they need to do their jobs.


Sources of Data

The source of the data is the EA tool, System Architect. The EA team built a set of openly available web services to provide the data to this portal and to other users who might need direct access to the EA data. Currently, the tool is available to all users within the GSA firewall.


Technical Overview

  • AngularJS- An open-source web application framework maintained by Google to address the challenges of developing single page web applications
  • Web Service Framework- A standard set of services, accessible through the web that provide data from System Architect in JSON (Javascript Object Notation) format
  • Bootstrap- An open-source set of tools designed to facilitate the creation of websites and web applications
  • D3 (Data Driven Documents)- An open-source Javascript library for manipulating documents (web pages) to create advanced visualizations of data

EA Web Service Description

The EA team has built a web service, accessible within the GSA firewall, that provides authoritative data from System Architect. The service powers this web application, but can also be used by other processes or applications. All service responses are in JSON format. The description of the service and the data that it can provide can be found here. The data sets available through this service include:

Frequently Asked Questions

Q: I am an owner of a system and I see information that is incorrect. Can I modify the information within this web application?

A: At this time there is no way to modify the data in System architect from the web application. If you see information that is incorrect you can gain access to the editing tool SA/XT or you can notify the EA team through the “Contact Us” link and we can make a change for you.

Q: I need to generate a report for my job. Is there any way to modify a report to suit my specific needs?

A: At this time, there is no ability to create custom reports in the tool. You can, however, export a report to one of many formats and customize the output in any way that suits you. You can also access the core data services in JSON (Javascript Object Notation) to make use in a custom application.
